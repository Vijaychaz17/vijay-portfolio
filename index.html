About Me

I am a Microsoft Cybersecurity Architect and Security Consultant with over 7 years of experience delivering enterprise security solutions across Microsoft Defender XDR, Microsoft Defender for Endpoint, Microsoft Defender for Cloud, Microsoft Sentinel, Microsoft Purview, and Azure Security.

I have successfully implemented end-to-end Microsoft Defender for Endpoint (MDE) and Microsoft Defender for Cloud (MDC) deployments across multiple enterprise environments using Microsoft Intune, SCCM (MECM), and Group Policy. My expertise includes endpoint security, cloud security posture management, vulnerability remediation, compliance management, and security automation.

I have led large-scale migrations of Windows workstations and servers from McAfee/Trellix to Microsoft Defender Antivirus, implementing EDR policies, Attack Surface Reduction (ASR) rules, security baselines, and device management strategies through Microsoft Intune and Microsoft Entra ID.

My experience includes onboarding and managing Windows and Linux servers within Microsoft Defender for Cloud across Azure, AWS, and hybrid infrastructures. I have successfully implemented Azure Arc onboarding, Defender for Servers, vulnerability management, and Secure Score improvement initiatives while driving remediation of security recommendations and compliance findings.

I have deployed Microsoft Sentinel SIEM/SOAR solutions across multiple customer environments, creating analytics rules, KQL queries, workbooks, playbooks, and incident response workflows. I have also integrated Microsoft Defender XDR components, including Defender for Endpoint, Defender for Identity, Defender for Office 365, and Defender for Cloud, providing centralized security visibility and incident correlation.

I am passionate about helping organizations strengthen their security posture through modern Microsoft Security technologies, operational excellence, and industry best practices.
